# 类加载器
- 启动类(或根类)加载器(Bootstrap ClassLoader)
- 扩展类加载器(ExtClassLoader)
- 应用类加载器(AppClassLoader)

# 类加载器的初始化
除启动类加载器外，扩展类加载器和应用类加载器的初始化都是在`Launcher`类中完成的，而Launcher类则由根类加载器进行加载。
